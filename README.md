# slimejs

`circom+snarkjs` libs w/o any node deps for full browser compat

---

`npm install`

`npm test`

`npm run fmt`

---

Main exports:

```js
export {
    MerkleTree,
    groth16FullProve,
    buildPoseidon,
    stringifyBigInts,
    hex
}
```
